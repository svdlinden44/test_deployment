# The Distillist

The Art of Fine Cocktails — curated recipes, the world's finest bars, and a personal cabinet built around what you love.

## Repository layout

- **`frontend/`** — Vite + React app (`npm install` and `npm run dev` from that directory).
- **`backend/`** — Django API and admin.

For **Railway** (Railpack at repo root): a minimal **`package.json`** and **`railway.toml`** at the repository root only forward `build` / `start` into `frontend/`. They are not used for day-to-day local frontend development.
